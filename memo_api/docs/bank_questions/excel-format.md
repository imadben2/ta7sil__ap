# تنسيق ملف Excel للاستيراد

## بنية الملف

يتكون قالب Excel من ورقة عمل رئيسية تحتوي على جميع الأسئلة بأنواعها المختلفة.

## الأعمدة المطلوبة

### 1. question_text_ar
**الاسم**: نص السؤال
**النوع**: نص
**مطلوب**: نعم ✅
**الحد الأقصى**: 1000 حرف

**مثال**:
```
ما هو ناتج 5 + 3؟
```

### 2. question_type
**الاسم**: نوع السؤال
**النوع**: نص محدد
**مطلوب**: نعم ✅

**القيم المقبولة**:
- `single_choice` أو `mcq_single` - اختيار من متعدد (خيار واحد)
- `multiple_choice` أو `mcq_multiple` - اختيار من متعدد (عدة خيارات)
- `true_false` - صح أو خطأ
- `short_answer` - إجابة قصيرة

**مثال**:
```
single_choice
```

### 3. option_1, option_2, option_3, option_4
**الاسم**: خيارات الإجابة
**النوع**: نص
**مطلوب**: فقط لأسئلة الاختيار من متعدد
**ملاحظة**: يجب تعبئة خيارين على الأقل

**مثال**:
```
option_1: 8
option_2: 7
option_3: 9
option_4: 10
```

### 4. correct_answer
**الاسم**: الإجابة الصحيحة
**النوع**: يعتمد على نوع السؤال
**مطلوب**: نعم ✅

**التنسيق حسب نوع السؤال**:

#### لأسئلة الاختيار من متعدد (خيار واحد):
يمكنك استخدام أي من الصيغ التالية:
- رقم الخيار: `1`, `2`, `3`, `4`
- حرف: `A`, `B`, `C`, `D`
- نص الخيار الكامل: `8`

**مثال**:
```
1
```
أو
```
A
```
أو
```
8
```

#### لأسئلة الاختيار من متعدد (عدة خيارات):
افصل الإجابات بفاصلة:
- أرقام: `1,3` أو `1,2,4`
- حروف: `A,C` أو `A,B,D`
- نصوص: `8,10` (نصوص الخيارات)

**مثال**:
```
1,3
```
أو
```
A,C
```

#### لأسئلة صح أو خطأ:
- `true` أو `صحيح` أو `1` للإجابة الصحيحة
- `false` أو `خطأ` أو `0` للإجابة الخاطئة

**مثال**:
```
true
```
أو
```
صحيح
```

#### لأسئلة الإجابة القصيرة:
اكتب الإجابة النموذجية مباشرة

**مثال**:
```
ثمانية
```

### 5. points
**الاسم**: النقاط
**النوع**: رقم صحيح
**مطلوب**: نعم ✅
**القيمة الافتراضية**: 1
**الحد الأدنى**: 1

**مثال**:
```
2
```

### 6. difficulty
**الاسم**: مستوى الصعوبة
**النوع**: نص محدد
**مطلوب**: لا (اختياري)

**القيم المقبولة**:
- `easy` أو `سهل`
- `medium` أو `متوسط`
- `hard` أو `صعب`

**مثال**:
```
easy
```
أو
```
سهل
```

### 7. explanation_ar
**الاسم**: الشرح
**النوع**: نص
**مطلوب**: لا (اختياري)
**الوصف**: شرح يظهر بعد الإجابة

**مثال**:
```
ناتج جمع 5 + 3 هو 8 لأن 5 زائد 3 يساوي 8.
```

### 8. tags
**الاسم**: الوسوم
**النوع**: نص مفصول بفواصل
**مطلوب**: لا (اختياري)

**مثال**:
```
رياضيات,جمع,أرقام
```

### 9. keywords
**الاسم**: الكلمات المفتاحية
**النوع**: نص مفصول بفواصل
**مطلوب**: لا (اختياري لأسئلة الإجابة القصيرة فقط)
**الوصف**: كلمات مفتاحية للتحقق من الإجابة

**مثال**:
```
ثمانية,8,eight
```

### 10. question_image_url
**الاسم**: رابط صورة السؤال
**النوع**: URL
**مطلوب**: لا (اختياري)

**مثال**:
```
https://example.com/images/question.jpg
```

## مثال كامل للصفوف

|-----------------|---------------|----------|----------|----------|----------|----------------|--------|------------|----------------|------|
| ما هو ناتج 5 + 3؟ | single_choice | 8 | 7 | 9 | 10 | 1 | 1 | easy | ناتج الجمع هو 8 | رياضيات,جمع |

### مثال 2: اختيار من متعدد (عدة خيارات)

|-----------------|---------------|----------|----------|----------|----------|----------------|--------|------------|
| اختر الأعداد الزوجية | multiple_choice | 2 | 3 | 4 | 5 | 1,3 | 2 | medium |

### مثال 3: صح أو خطأ

| question_text_ar | question_type | correct_answer | points | explanation_ar |
|-----------------|---------------|----------------|--------|----------------|
| الشمس نجم | true_false | true | 1 | الشمس نجم لأنها تنتج الضوء والحرارة |

| question_text_ar | question_type | correct_answer | points | keywords |
|-----------------|---------------|----------------|--------|----------|
| ما هي عاصمة مصر؟ | short_answer | القاهرة | 1 | القاهرة,cairo |

## نصائح مهمة

### ✅ افعل

- استخدم UTF-8 لحفظ الملف لدعم العربية
- راجع جميع الأسطر قبل الرفع
- استخدم تنسيق نص لجميع الأعمدة
- احتفظ بنسخة احتياطية من الملف

### ❌ لا تفعل

- لا تترك خلايا فارغة للحقول المطلوبة
- لا تستخدم أحرف خاصة في رؤوس الأعمدة
- لا تحذف أو تعيد تسمية الأعمدة
- لا تضع أكثر من نوع بيانات في نفس العمود

## التحقق من الصحة

قبل رفع الملف، تأكد من:

1. ✅ جميع الحقول المطلوبة مملوءة
2. ✅ أنواع الأسئلة صحيحة
3. ✅ الإجابات الصحيحة بالتنسيق الصحيح
4. ✅ النقاط أكبر من أو تساوي 1
5. ✅ لا توجد أسطر فارغة بين الأسئلة

## أدوات مساعدة

### Google Sheets
يمكنك استخدام Google Sheets لملء القالب ثم تنزيله بصيغة Excel.

### Microsoft Excel
استخدم Excel 2016 أو أحدث لضمان التوافق الكامل.

### LibreOffice Calc
يمكن استخدامه أيضاً، لكن تأكد من حفظ الملف بصيغة .xlsx
