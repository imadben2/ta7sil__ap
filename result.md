# دورة تحفيض - Course Import Script

## Instructions

1. Copy the seeder file below to: `memo_api/database/seeders/TahfidCourseSeeder.php`
2. Run the seeder: `php artisan db:seed --class=TahfidCourseSeeder`
3. After running, upload videos and update `video_url` fields in the database

            CourseLesson::create([
                'course_module_id' => $module->id,
                'title_ar' => $title,
                'description_ar' => $title,
                'order' => $order++,
                'content_type' => 'video',
                'video_type' => 'upload',
                'video_url' => null, // Placeholder - upload later
                'video_duration_seconds' => 0,
                'is_free_preview' => false,
                'is_published' => false,
            ]);
        }

        $this->command->info("  Created {$order} lessons in module: {$module->title_ar}");
        return count($lessonTitles);
    }

## Quick Alternative: SQL Insert Script

If you prefer raw SQL, run this after creating the course manually:

```sql
-- First, create the course and get its ID
-- Then use this pattern for modules and lessons

-- Example for one module:
INSERT INTO course_modules (course_id, title_ar, description_ar, `order`, is_published, created_at, updated_at)
VALUES (YOUR_COURSE_ID, 'شخصيات التاريخ', 'شخصيات تاريخية مهمة', 1, 0, NOW(), NOW());

-- Get the module ID, then insert lessons:
INSERT INTO course_lessons (course_module_id, title_ar, description_ar, `order`, content_type, video_type, is_published, created_at, updated_at)
VALUES

-- ... etc
```
